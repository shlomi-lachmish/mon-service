# Usage

<!-- usage -->
```sh-session
$ npm install -g mon-service
$ mon-service COMMAND
running command...
$ mon-service (-v|--version|version)
mon-service/0.0.7 darwin-x64 node-v13.7.0
$ mon-service --help [COMMAND]
USAGE
  $ mon-service COMMAND
...
```
